How it works

1. You put two PDFs in the data folder (Rename them to A_BASE and A_COMPARE), one is your base drawing, and the other is the updated version.

- The program converts both into .tif images.

- It compares them pixel by pixel:

- Green means something is missing compared to the base.

- Red means something extra was added.

- Black means both drawings match perfectly.

- It saves a new file called A_COMPARE.tif in the output folder showing all differences.
